Kinematics - Projectile Motion

psychology AI Summary

Projectile motion involves independent horizontal and vertical motions. The horizontal velocity remains constant, while vertical motion is subjected to constant acceleration due to gravity. The trajectory path is always a parabola.

Consider an object projected with an initial velocity u at an angle θ to the horizontal.

R = u² sin(2θ) / g

The maximum horizontal range occurs when sin(2θ) = 1, which means 2θ = 90° or θ = 45°.
